# Financial Data Modelling Portfolio

This repository contains my coursework portfolio for the University of London MSc Data Science module **DSM120 Financial Data Modelling**.

## Contents

- Time series exploratory analysis
- Stationarity analysis
- Trend and seasonal decomposition
- Log transformations
- Naïve, average and seasonal naïve forecasting
- Autoregressive (AR) models
- ACF and PACF analysis
- Automated trading simulation
- Monte Carlo simulation
- Prophet forecasting
- Forecast evaluation
